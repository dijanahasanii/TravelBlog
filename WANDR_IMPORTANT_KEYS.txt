--------------------------------------------------------------------------------
  7. WHAT HAPPENS IF YOU LOSE THESE
--------------------------------------------------------------------------------

    MongoDB password lost:
      -> Log in to cloud.mongodb.com > Database Access > Edit user > change password
      -> Update MONGO_URI in all 3 .env files

    JWT secrets lost:
      -> Generate new ones:  node -e "const c=require('crypto'); console.log(c.randomBytes(64).toString('hex'))"
      -> Update JWT_SECRET in all 3 .env files
      -> WARNING: All logged-in users will be logged out automatically

    VAPID keys lost:
      -> Generate new ones:  node -e "const wp=require('web-push'); console.log(wp.generateVAPIDKeys())"
      -> Update notification-service .env
